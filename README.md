# CRM-Projekt

Willkommen beim CRM (Customer Relationship Management)-Projekt! Diese einfache CRM-Anwendung ermöglicht das Verwalten von Kundendaten, einschließlich Vorname, Nachname, E-Mail und Telefonnummer. Sie können Kundeneinträge einfach erstellen, aktualisieren und löschen.

## Funktionen

- **Kunden erstellen:** Fügen Sie neue Kunden zum CRM hinzu, indem Sie Vorname, Nachname, E-Mail und Telefonnummer angeben.

- **Kunden aktualisieren:** Ändern Sie Kundendaten wie Vorname, Nachname, E-Mail und Telefonnummer. Änderungen werden sofort in der Datenbank aktualisiert.

- **Kunden löschen:** Entfernen Sie einen Kunden aus dem CRM. Seien Sie vorsichtig, da diese Aktion nicht rückgängig gemacht werden kann.

## Verwendete Technologien

- **Frontend:** React
- **Styling:** Bootstrap
- **Backend:** Node.js, Express
- **Datenbank:** MySQL
